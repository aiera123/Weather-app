Weather App ☀️🌧️

A simple and responsive weather application built using ReactJS.
Users can search for any city and view real-time weather information using the OpenWeather API.

Features ✨
Search weather by city name
Real-time weather data
Displays:
Temperature
Weather condition
City name
Modern responsive UI
API integration using Axios
Built with React Hooks (useState)

#Technologies Used 🛠️
ReactJS
CSS3
Axios
OpenWeather API

Installation 🚀

Clone the repository:

git clone https://github.com/aiera123/Weather-app.git


Install dependencies:

npm install

Start development server:

npm start
